# 사우회 회비 인터랙티브 웹 대시보드 프로젝트

## 📋 프로젝트 개요

사우회 회비 거래 내역을 투명하고 깔끔하게 모든 직원들이 볼 수 있도록 **인터랙티브 HTML 웹 대시보드**를 제작합니다.

---

## 🎯 프로젝트 목표

1. **투명성**: 모든 거래 내역을 누구나 쉽게 확인 가능
2. **시각화**: 차트와 그래프로 한눈에 파악
3. **접근성**: 웹 브라우저만 있으면 어디서든 확인 가능
4. **인터랙티브**: 검색, 필터링, 정렬 기능 제공
5. **모바일 대응**: 스마트폰에서도 깔끔하게 표시

---

## 📊 데이터 구조 및 배경

### 1. 데이터 소스
- **신한은행 거래내역**: `신한은행_거래내역조회_20251111111910.xls`
  - 2019년 4월 ~ 2020년 11월 (초기 운영 기간)
  - 2020년 12월부터 카카오뱅크로 완전 전환
  - **현재 잔액: 0원** (모두 카카오뱅크로 이전)

- **카카오뱅크 거래내역**: `251111_사우회회비_통장_거래_내역카카오뱅크계좌.xlsx`
  - 2021년 1월 ~ 2025년 11월 (현재)
  - 총 794건의 거래
  - **현재 통장 잔액: 399,204원**

### 2. 중요 개념: 세이프박스
- **세이프박스**: 카카오뱅크의 개인금고 기능 (이자 발생)
- 사우회 자금 중 일부를 세이프박스에 보관하며 운용
- 거래 내역:
  - 넣은 금액 (출금으로 표시): 91,800,000원
  - 꺼낸 금액 (입금으로 표시): 78,529,044원
  - 이자 수익: 279,044원 (25회)
  - **현재 세이프박스 보관액: 16,094,818원**

### 3. 최종 결산 (2025년 11월 11일 기준)
```
신한은행 잔액:           0원
카카오뱅크 통장 잔액:    399,204원
세이프박스 보관액:       16,094,818원
─────────────────────────────────
실제 운용 가능 총액:     16,494,022원
```

## 🎨 웹 대시보드 요구사항

### 필수 기능

#### 1. 메인 대시보드
- **현재 잔액 카드**
  - 카카오뱅크 통장: 399,204원
  - 세이프박스: 16,094,818원
  - 총 운용액: 16,494,022원 (강조)

- **주요 통계 카드**
  - 총 입금액 (누적)
  - 총 출금액 (누적)
  - 이자 수익
  - 총 거래 건수

#### 2. 차트 및 시각화
- **연도별 수입/지출 막대 그래프**
- **월별 거래 추이 선 그래프**
- **수입 vs 지출 원형 차트**
- **세이프박스 입출금 추이**

#### 3. 거래 내역 테이블
- **필터링 기능**
  - 은행별 (신한은행 / 카카오뱅크)
  - 구분별 (입금 / 출금)
  - 날짜 범위
  - 세이프박스 포함/제외 토글

- **검색 기능**
  - 내용 검색 (예: "이동혁", "세이프박스")

- **정렬 기능**
  - 날짜순, 금액순

- **페이지네이션**
  - 한 페이지당 20~50건

#### 4. 세이프박스 상세 페이지
- 세이프박스 입출금 내역 전체
- 이자 수익 내역
- 현재 보관액 강조
